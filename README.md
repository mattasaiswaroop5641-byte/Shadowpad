ShadowPad is a real-time, end-to-end encrypted collaborative workspace. It allows users to create secure rooms, edit text together in real-time, and share files securely.

## Features

- **Real-time Collaboration**: Live text syncing between multiple users.
- **Secure Rooms**: Password-protected rooms with ephemeral storage (data is cleared when the room is empty).
- **File Sharing**: Drag-and-drop file sharing (up to 25MB).
- **Host Permissions**: Room creators can toggle editing, uploading, and deleting permissions for other users.
- **User Management**: Live user list with host migration support.

## How to Run

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start the Server**:
   ```bash
   npm start
   ```

3. **Open in Browser**: Visit `http://localhost:3000`
