üîê Security & Encryption (The Core)
End-to-End Encryption (E2E):
Algorithm: Uses AES-256-GCM for message encryption.
Protocol: Implements a Double Ratchet algorithm (similar to Signal/WhatsApp) for forward secrecy.
Key Derivation: Uses PBKDF2 (100,000 iterations) to derive keys from the room password, followed by HKDF for chain keys.
Admin Blindness: The server only stores encrypted blobs (ciphertext, iv, salt, step). It cannot read the content.
Verification: A "Verify Encryption" tool in settings allows users to audit the cryptographic proof and latency in real-time.
Hybrid Storage System:
MongoDB Atlas: Persists encrypted data for up to 30 days.
Memory Fallback: Automatically switches to RAM-only mode if the database connection fails, ensuring the app never goes down.
Auto-Destruction: Pads are automatically deleted from the database if not accessed for 30 days (TTL index).
